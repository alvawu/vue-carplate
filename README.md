# vue-carplate

> vue中国车牌号录入组件, Chinese Car Plate Component For Vue


## install plugin
``` bash
npm install vue-carplate --save
or
yarn add vue-carplate
```

## import plugin
``` bash
import Vue from 'vue'
import CarPlate from 'vue-carplate'
Vue.use(CarPlate)
```
## usage

``` bash
<car-plate v-model="yourCarPlateVariable"></car-plate>

```
